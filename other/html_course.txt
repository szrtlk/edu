РАБОТА С HTML

КОММЕНТАРИЙ К КУРСУ
* код в виде столбца будет здесь для удобства отображаться одной строкой, новая строка обозначается знаком /
* пример: const firstName = 'Pavel' / const lastName = 'Ivanov' / age = 35
* результат работы программы будет здесь отображаться после знака =>

ТЕГИ
* теги содержатся в <>
* теги бывают открывающие (<tag>), закрывающие(</tag>) и одинарные (<tag/>)
* если тег будет одинарным, это будет указано отдельно

АТРИБУТЫ
* атрибут - свойство тега
* пример атрибутов: charset, title
* пример использования атрибутов: <meta charset="UTF-8"/>, <abbr title="Hypertext Markup Language">
* атрибут указывается в открывающем или одиночном теге
* значение атрибута указывается в "" после знака =

ОТОБРАЖЕНИЕ НЕСТАНДАРТНЫХ СИМВОЛОВ
* < - &lt;
* > - &gt;
* & - &amp;
* " - &quot;

КОММЕНТАРИЙ В КОДЕ
* пример комментария - <!-- Комментарий -->

ВАЛИДАТОР HTML
* для валидации HTML используется валидатор <!DOCTYPE html>

ОСНОВНЫЕ ТЕГИ
* <html> - информация о наличии кода HTML
* <head> - информация о заголовке странице
* <body> - содержимое странице

ТЕГИ РАЗДЕЛА <HEAD>
* <meta> - метаданные, *одиночный тег*
* <meta charset="UTF-8"/> - информация о кодировке 
* <title> - заголок страницы

ТЕГИ РАЗДЕЛА <BODY>
* <h1> - заголовок 1 уровня
* <h2> - заголовок 2 уровня 
* <h3> - заголовок 3 уровня
* <h4> - заголовок 4 уровня
* <h5> - заголовок 5 уровня
* <h6> - заголовок 6 уровня, меньше обычного текста
* <p> - обычный текст
* <br/> - перенос текста, *одиночный тег*
* <hr/> - черта, *одиночный тег*

ТЕГИ ФОРМАТИРОВАНИЯ
* <pre> - текст с сохранением всех символов форматирования, включая табуляции, пробелы и переносы строк
* <small> - маленький текст
* <b> - жирный текст
* <i> - курсивный текст
* <u> - подчёркнутый текст
* <s> - зачёркнутый текст
* <strong> - жирный текст, логическое выделение
* <em> - курсивный текст, логическое выделение
* <ins> - подчёркнутый текст, пометка текста как добавленного вместо неверного
* <del> - зачёркнутый текст, пометка текста как неверного
* <sub> - текст в нижнем регистре
* <sup> - текст в верхнем регистре
* <q> - цитата, текст в кавычках
* <blockquote> - цитата, текст в отдельном блоке
* <samp> - текст в виде примера
* <bdo dir="rtl"> - текст справа налево

ПРОЧИЕ ТЕГИ ФОРМАТИРОВАНИЯ
* <code> - текст, представляющий компьютерный код
* <var> - текст, представляющий переменную
* <kbd> - текст, который введён с клавиатуры
* <abbr> - текст аббревиатуры
* <abbr title="HyperText Markup Language">HTML</abbr> - пример аббревиатуры с расшифровкой при наведении
* <address> - текст адреса, отображается курсивом

ССЫЛКИ
* <a> - ссылка
* внутри тега задаётся название ссылки, а адрес ссылки задаётся в атрибуте href
* пример - <a href="https://www.google.com">Google</a>
* атрибут target="_blank" загружает страницу в новом окне

ЯКОРЯ
* якорь задаётся с помощью атрибута id, например, <h1 id="part1">Часть 1</h1>
* ссылка на якорь задаётся как обычная ссылка, но вместо внешней ссылки указывается название якоря после #, например, <a href="#part1">Часть 1</a>
* ссылку можно задать на якорь на другой странице, для этого нужно указать сначала ссылку на другую страницу, потом / , потом # и название якоря

АУДИО
* <audio> - добавление аудио
* атрибут controls - добавляет панель управления к аудиофайлу, например, controls="controls"
* атрибут autoplay - звук начинает играть сразу после загрузки страницы
* атрибут loop - устанавливает, сколько раз будет проигрываться музыка; по умолчанию проиграется 1 раз
* атрибут src - путь к воспроизводимому файлу
* вместо атрибута src можно использовать атрибут-тег <source/>, *одиночный тег*, например, <source src="music1.mp3" type="audio/ogg; codecs=vorbis" />, данный код позволяет перекодировать аудиофайл в другой тип аудио

ИЗОБРАЖЕНИЯ
* <img/> - добавление изображения, *одиночный тег*
* атрибут src - путь к изображению
* атрибут alt - описание изображения в случае, если оно не загрузится
* атрибут width - ширина изображения, указывается в пикселях или % к размеру к окну браузера (тогда указывается, например 100%)
* атрибут height - высота изображения, указывается в пикселях или % к размеру к окну браузера (тогда указывается, например 100%)
* атрибут title - текст подсказки при наведении на картинку

СПИСКИ
* <ol> - упорядоченный список, элементы списка задаются атрибутом-тегом <li>
* <ul> - неупорядоченный список, элементы списка задаются атрибутом-тегом <li>
* <dl> - список определений - список терминов и определений терминов, термины задаются атрибутом-тегом <dt>, определения задаются атрибутом-тегом <dd>

ТАБЛИЦЫ
* <table> - добавление таблицы
* по умолчанию таблицы создаются без границ
* атрибут border задаёт толщину границ, например, borders="1"
* атрибут caption задаёт название таблицы
* строки внутри таблицы задаются атрибутом-тегом <tr>
* ячейки задаются внутри строки атрибутом-тегом <td>
* для выделения первой строки или первого столбца можно вместо атрибута-тега <td> использовать атрибут-тег <th>
* для отображения пустой ячейки можно указывать символ -, в некоторых браузерах ячейки без значения могут не отобразиться
* для объединения ячеек таблицы используются атрибуты rowspan и colspan атрибута <td>, например, rowspan="2" растягивает ячейку вниз, colspan="2" растягивает ячейку вправо, в обоих случаях возникает объединённая ячейка

ПРИМЕР ТАБЛИЦЫ
<table border="1">
<tr> <td></td> <th> Столбец 1 </th> <th> Столбец 2 </th> </tr>
<tr> <th> Строка 1 </th> <td> Строка 1, столбец 1 </td> <td> Строка 1, столбец 2 </td> </tr>
<tr> <th> Строка 2 </th> <td> Строка 2, столбец 1 </td> <td> Строка 2, столбец 2 </td> </tr>
</table>

СТИЛИ ТАБЛИЦЫ
* также можно выделить строки таблицы в группы, с различным функциональным назначением и назначить им разные стили
* можно выделить заголовочный часть <thead>, основную часть <tbody> и итог <tfoot>
